## ✨ About This Project
This is a sleek, responsive Weather App User Interface (UI) designed to mimic the iPhone 13 Pro Max screen. It is built completely from scratch using HTML, CSS, and vanilla JavaScript.

## 🌟 Features
- **iPhone Frame Layout:** Displays an interactive, heavily styled iPhone mockup on desktop devices.
- **Responsive Design:** Automatically adapts to a full-screen native app experience on mobile screens.
- **Dynamic Data Rendering:** Click the search icon and enter cities (e.g., London, Dhaka, New York) to update weather parameters dynamically.
- **Live Date:** Automatically fetches and formats the current local date.

## 🛠️ Technologies Used
- **HTML5** - For the structural markup.
- **CSS3** - For styling, gradients, flexbox layouts, and media queries.
- **JavaScript (Vanilla)** - For DOM manipulation, state management, and real-time updates.
- **Google Fonts** - Featuring the 'Inter' typeface for a clean, Apple-like aesthetic.
- **FontAwesome** - For scalable vector icons.

## 🚀 How to Run
1. Create a new folder (e.g., `WeatherApp`).
2. Add the `index.html`, `style.css`, and `script.js` files into this folder.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.).
